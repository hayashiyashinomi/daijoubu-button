# daijoubu-button

## サービス概要
ボタンを押すだけで、各国語の「大丈夫（It’s OK / It’s gonna be fine）」を表示します。
不安なとき、自分や周りにポジティブな一言を届ける“即効性”に特化したシンプルなサービスです。
MVPは表示・切替・コピーのシンプルな構成とし、今後読み上げやSNS共有などを追加していきたいです。

## このサービスへの思い・作りたい理由
卒業日（11/5）まで時間が限られており、「小さく作って確実にリリースする」経験を得たいと考えました。
また実際に学習を進める中で、不安や焦りを感じることが多かったため、
押すだけですぐに安心できるツールが自分にも必要だと感じました。
追い込み時期に寄り添うようなアプリを、自分のためにも誰かのためにも作りたいです。

## ユーザー層について
- 学習や制作で不安を抱えやすい受講生や初学者
- 日常生活で気持ちを切り替えたい人
- 多言語コミュニティや海外生活者

**理由：**
ボタンひとつで安心できるシンプルな仕組みは、余裕がない状況でも負担なく利用できるためです。

## サービスの利用イメージ
Webページにアクセス
- 画面中央の大きなボタンを押す
- ランダムに表示される「大丈夫」（多言語版）を見る
- 必要に応じてコピーしてSNSやチャットに共有
  (将来的には読み上げ・カード形式共有機能も追加予定）

## ユーザーの獲得について
- X（旧Twitter）やLINEでの共有導線（OGPカード）を設定
- 学習コミュニティや受講生仲間への共有
- 簡易なQRコードを作成し、スマホでもアクセスを促す

## サービスの差別化ポイント・推しポイント
- 1アクションで完結する圧倒的シンプルさ
- 他言語の「大丈夫」により好奇心と安心の両方を提供
- 負担がかからず、疲れているときでも使える
- MVP完成後も拡張性が高い（音声・共有・履歴・PWA 等）

## 機能候補
- MVP（卒業制作で実装）
- ボタン押下で「大丈夫」表示
- 多言語メッセージ（12〜20言語、YAML・配列ハードコード）
- ランダム or 次へ切り替え
- クリックでコピー
- シンプルなUI／ダークモード

## 今後の機能
- 音声読み上げ（Web Speech API / TTS）
- SNSカード画像生成
- お気に入り登録
- 翻訳追加UI
- PWA対応（オフライン利用）

## 使用する技術スタック
- フレームワーク：Ruby on Rails 7.x（Hotwire / Stimulus）
- 言語：Ruby 3.x
- DB：MVPは不要（必要であればPostgreSQL）
- デプロイ：Render / Fly.io（無料枠想定）
- CSS：Tailwind CSS
- テスト：RSpec（最低限のSystemテスト）
- その他：i18n / dotenv-rails
- 実装イメージ
- 多言語リストをconfig/locales/messages.ymlに保持
- コントローラーで1件取得→View表示
- ボタンイベントはStimulusで制御
- クリップボードはnavigator.clipboard.writeTextを使用
