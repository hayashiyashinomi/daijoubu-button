# daijoubu-button

## サービス概要
ボタンを押すだけで、各国語の「大丈夫（It’s OK / It’s gonna be fine）」を表示します。  
不安なとき、自分や周りにポジティブな一言を届ける“即効性”に特化したシンプルなサービスです。

本サービスは UX 検証を目的とした MVP として、
表示・切替・コピーに機能を絞って公開しています。  
今後は読み上げや SNS 共有などの拡張を予定しています。

---

## このサービスへの思い・作りたい理由
卒業制作の期限が迫る中で「小さく作って確実にリリースする」経験を得たいと考えました。

また学習を進める中で不安や焦りを感じる場面が多く、
押すだけですぐに安心できるツールが自分にも必要だと感じたことが開発のきっかけです。

---

## ユーザー層について
- 学習や制作で不安を抱えやすい受講生や初学者
- 日常生活で気持ちを切り替えたい人
- 多言語コミュニティや海外生活者

**理由：**  
ボタンひとつで完結する仕組みは、余裕がない状況でも負担なく利用できるためです。

---

## サービスの利用イメージ
- Webページにアクセス
- 画面中央の大きなボタンを押す
- ランダムに表示される「大丈夫」（多言語）を見る
- 必要に応じてコピーして SNS やチャットに共有  
（将来的には読み上げ・カード形式共有機能を追加予定）

---

## ユーザーの獲得について
- X（旧Twitter）や LINE での共有導線（OGPカード）
- 学習コミュニティでの紹介
- QRコード配布によるスマホ導線

---

## サービスの差別化ポイント
- 1アクションで完結する UX
- 多言語対応による安心感と好奇心の両立
- 疲れている時でも使える設計
- MVP後の高い拡張性（音声・共有・履歴・PWA 等）

---

## 機能

### MVP（実装済）
- ボタン押下でメッセージ表示
- 多言語対応（YAML 管理）
- ランダム切り替え
- コピー機能
- ダークモード

### 今後の機能
- 音声読み上げ（Web Speech API / TTS）
- SNSカード画像生成
- お気に入り登録
- 翻訳追加UI
- PWA対応

---

## 使用技術

- フレームワーク：Ruby on Rails 8.x
- 言語：Ruby 3.x
- DB：MVPでは未使用（YAML 管理）／将来的に PostgreSQL
- CSS：Tailwind CSS
- デプロイ：Render
- テスト：RSpec
- その他：i18n / GitHub Actions / dotenv-rails

---

## CI / 品質管理

GitHub Actions による CI を導入しています。

Pull Request 作成時および手動実行時に以下をチェックします。

- Brakeman（Rails セキュリティ解析）
- bundler-audit（Gem 脆弱性検査）
- importmap audit（JS 依存関係チェック）

---

## デプロイ・運用

- Render にデプロイ済み
- 無料枠スリープ対策実施
- 本番トラブル時の確認フローを整備

### 表示不具合時の確認手順
- ブラウザのキャッシュクリア
- DevTools → Network 確認
- GitHub Actions の CI 状況確認
